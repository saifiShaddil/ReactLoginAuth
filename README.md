This is the Login authentication app

Clone down this repository. You will need node and npm installed globally on your machine.

1). First download the project or clone the project to your computer.

2). Open the project in your text editor ( atom, vscode).

3). Open the terminal and make the this project your root project, and run => 
        npm install
this command will install all the required packages required for the working of this project

4). Finally, just type => 
        npm start , run this in your terminal 

if every thing is done correctly then you will be redirectly to your web browser at localhost:3000 , where you can see the project view.
